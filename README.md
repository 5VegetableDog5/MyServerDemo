# MyServerDemo
一个服务器，按照规定的协议可用作为客户端的桥接，以使客户端进行通信  
V1.1 更新说明：

1. 支持当某个客户端处于接听状态时，主动断开与对方的通话;
1. 修复了一些问题，如在接收时接收的类改为QByteArray类，以确保音频数据正常
